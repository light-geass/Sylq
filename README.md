# Sylq (GATER)

Sylq is a comprehensive platform for AI-powered test generation and analysis. It helps users create tests, track history, and analyze performance with an intuitive interface.

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS
- **Backend**: FastAPI (Python)
- **Authentication**: Firebase Authentication
- **Database**: Supabase (PostgreSQL)

## Project Structure

- `/frontend`: Next.js application with React components and modern styling.
- `/backend`: FastAPI server handling logic, AI integration, and database operations.
- `/Questions_ingestion`: Utilities for ingesting and processing test questions.

## Getting Started

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

### Backend
1. Navigate to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Start the server: `uvicorn main:app --reload`
